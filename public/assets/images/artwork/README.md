# Artwork Images

Drop exported JPG, PNG, or WebP files for the artwork section in this folder.

Suggested filenames currently referenced by the site:

- `painting-01.jpg`
- `sculpture-01.jpg`
- `process-01.jpg`

For now, this static website cannot permanently upload images from the browser.
To publish a photo, add the image file here, update the matching `<img src="">`
path in `index.html` if needed, then commit and push the change.
